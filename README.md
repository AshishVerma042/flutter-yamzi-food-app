# 🍱 Yamzi - Custom Thali App

A Flutter-based food ordering application where users can customize their thali, select location, and place orders easily.

---

## 🚀 Features
- 🍽️ Custom Thali (Veg & Non-Veg)
- 📍 Location & Map Integration
- 🛒 Order Placement
- 👤 User Profile
- 🔐 Authentication (Login/Signup)
- 📦 Order History

---

## 🛠️ Tech Stack
- Flutter
- Firebase Auth
- Cloud Firestore
- Google Maps API

---

## ⚙️ Installation
bash
- git clone https://github.com/your-username/flutter-yamzi-food-app.git
- cd yamzi-thali-app
- flutter pub get
- flutter run
